# Task-001 Security Instruction

## Goal
env-marker の脅威モデルを作成し、初期対策を提示する。

## Steps
1. 資産（設定値、判定ルール、表示状態）を列挙。
2. 攻撃面（content script, options, storage, message passing）を整理。
3. 脅威を最低10件列挙し、影響度と確率を評価。
4. それぞれに具体的対策を割り当てる。

## Output
`security/deliverables/task-001-threat-model.md`

## Required Sections
- Threat list table
- Mitigations
- Residual risks
